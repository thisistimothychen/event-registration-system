# Event Registration System 1.0
###### Generic enterprise event registration system with Google Calendar integration


## Database Options
### Events
- Name
- Location
- Start Date
- End Date
- Start Time
- End Time
- Description
- Registration Limit

### Users
(look into using UMD's CAS system API)
- First Name
- Last Name
- Email Address
- Username
- User Privileges
